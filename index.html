Projeto URI

C, Lógica de Programação

URI é um projeto do grupo PET BCC que tem como objetivo resolver problemas da plataforma URI online judge que contêm mais de 3 mil problemas de diferentes dificuldades.
Com intuito de auxiliar estudantes iniciantes em programação todos os problemas resolvidos são disponibilizados no github junto com um README em markdown com uma explicação didática da lógica utilizada para resolver o problema, com trechos do código e links auxiliares.
Liderei o início do projeto, até o término de todos exercícios da categoria Iniciante.
